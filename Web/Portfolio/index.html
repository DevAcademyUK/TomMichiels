Week 1 Java Basics

The first week of the course was focused on covering the very basics of Java. During this week we worked on several small programs, culminating with a basic command-line banking application that allowed users to access multiple accounts within a user; withdrawing, depositing and transferring funds from each.

My version of this code can be seen by following the GitHub link below, along with the Problem Definition Statement, Class Diagram and Data Dictionary created for the project.

The problem to be tackled is to create a simple banking program. This should support access blocked via a PIN. The user should have access to multiple accounts (current, saving, joint). Accounts should be able to store and display their balance. Accounts should facilitate depositing and withdrawal to and from their balance. The user should be able to transfer funds directly from one account to another.

Week 2 HTML, CSS and Bootstrap

The second week was based on developing a basic level of understanding with front-end website creation, touching briefly on HTML, CSS and on the Bootstrap framework (that is being used to create this portfolio).

There were two main projects of the week - the first being this portfolio, and the second being a dummy e-commerce site, featuring multiple pages, interactive elements (utilising js), and compliance with w3c standards. We were also tasked with creating a site flow diagram for said e-commerce site.
